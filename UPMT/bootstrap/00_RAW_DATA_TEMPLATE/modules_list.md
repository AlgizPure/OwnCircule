## ТЕХНОЛОГИЧЕСКИЙ СТЕК

### Frontend (Mobile)
- **Framework:** React Native 0.73+
- **State:** Redux Toolkit + RTK Query
- **Navigation:** React Navigation 6
- **UI:** React Native Paper (Tiffany-стиль)
- **QR:** react-native-vision-camera
- **Push:** Firebase Cloud Messaging
- **Analytics:** Firebase Analytics + Amplitude
- **Crash:** Sentry

### Backend
- **Language:** Python 3.11+
- **Framework:** FastAPI 0.104+
- **Task Queue:** Celery + Redis
- **ORM:** SQLAlchemy 2.0
- **API Docs:** OpenAPI 3.0 (Swagger)
- **Auth:** JWT (python-jose)

### Databases
- **PostgreSQL 15:** Основные данные (пользователи, транзакции, мероприятия)
- **ClickHouse 23:** Аналитика, метрики, RFM
- **Redis 7:** Кеш, сессии, очереди Celery
- **Elasticsearch 8:** Полнотекстовый поиск

### Infrastructure
- **Cloud:** Yandex Cloud / VK Cloud (РФ)
- **Containers:** Docker + Docker Compose
- **CI/CD:** GitHub Actions
- **Monitoring:** Prometheus + Grafana, Sentry
- **Storage:** S3-совместимое хранилище

## КЛЮЧЕВЫЕ МЕТРИКИ

**North Star Metric:**
- NSM = % клиентов, совершивших ≥2 покупки в разных категориях за 60 дней
- Цель: 25% к концу 6-го месяца

**Операционные метрики:**
- CAC (Customer Acquisition Cost)
- LTV (Lifetime Value)
- Retention Rate
- Churn Rate
- NPS (Net Promoter Score)
- Cross-Purchase Rate
- Ecosystem Revenue Share

## СТАТУСЫ УЧАСТНИЦ

1. **Insider** (базовый): Регистрация + 1 покупка | Кешбэк 5%
2. **VIP**: ≥30,000₽ + ≥3 категории | Кешбэк 7%
3. **Elite**: ≥100,000₽ + ≥5 категорий | Кешбэк 10%
4. **Inner Circle**: По приглашению | Амбассадор
