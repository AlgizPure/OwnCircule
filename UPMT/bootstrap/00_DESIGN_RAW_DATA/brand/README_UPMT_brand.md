- Variations (light bg, dark bg)

---

## TIPS

1. **SVG предпочтительнее** PNG (масштабируется)
2. **Документируй usage** - где какой logo использовать
3. **Brand ≠ UI colors** - это разные вещи!
4. **Если бренда нет** - не проблема, сфокусируйся на UI
