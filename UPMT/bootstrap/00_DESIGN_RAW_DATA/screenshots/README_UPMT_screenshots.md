# SCREENSHOTS

Скриншоты продуктов-вдохновений и референсов.

---

## ЧТО КЛАСТЬ

- Скриншоты конкурентов
- Примеры UI которые нравятся (Linear, Notion, Figma...)
- Конкретные UI patterns
- `notes.md` - общие комментарии

---

## NAMING

Называй файлы описательно:

```
✅ Хорошо:
linear-dashboard.png
notion-sidebar-navigation.png
figma-component-panel.png

❌ Плохо:
screenshot1.png
image.png
photo_2025.png
```

---

## NOTES.MD

Создай `notes.md` с комментариями:

```markdown
# Screenshot Notes

## linear-dashboard.png
**Что нравится:**
- Скорость (instant feedback)
- Keyboard shortcuts visible
- Clean layout, много white space
- Subtle animations

**Что применить:**
- Keyboard-first approach
- Minimal UI
- Fast interactions

## notion-sidebar-navigation.png
**Что нравится:**
- Collapsible sections
- Clear hierarchy
- Search at top
- Recent pages

**Что применить:**
- Similar navigation structure
- Quick search
```

---

## TIPS

1. Делай скриншоты конкретных фич, не целых страниц
2. Объясняй ЧТО именно нравится
3. Указывай ЧТО хочешь применить у себя
4. Сравнивай (A лучше чем B, потому что...)

---

**Минимум:** 3-5 скриншотов с notes
