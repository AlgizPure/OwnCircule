## Versioning

**Semantic Versioning (semver):**

- **Major (X.0.0):** Breaking changes
- **Minor (0.X.0):** New features (backward compatible)
- **Patch (0.0.X):** Bug fixes

**Examples:**
- New component: Minor (1.1.0)
- Component API change: Major (2.0.0)
- Bug fix: Patch (1.0.1)
