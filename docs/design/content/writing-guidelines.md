## üìã WRITING CHECKLIST

Before finalizing any copy, verify:

**Clarity:**
- [ ] Can a busy person understand this in 5 seconds?
- [ ] Are there unfamiliar terms? (Define or replace them)
- [ ] Is the main point clear in the first sentence?

**Conciseness:**
- [ ] Can any words be deleted? (Delete them)
- [ ] Can any phrase be replaced with fewer words?
- [ ] Is every word earning its place?

**Specificity:**
- [ ] Are there vague words like "–º–Ω–æ–≥–æ", "—Å–∫–æ—Ä–æ", "–º–æ–∂–µ—Ç –±—ã—Ç—å"?
- [ ] Does this tell the user exactly what to expect?
- [ ] Are numbers specific?

**Action:**
- [ ] Does this use strong verbs?
- [ ] Is it focused on what the user gets, not system processes?
- [ ] Does it tell them what to do next?

**Tone:**
- [ ] Would we say this to a friend we respect?
- [ ] Is it warm without being overly familiar?
- [ ] Does it respect her intelligence and time?

**Format:**
- [ ] Is capitalization correct (sentence case)?
- [ ] Are buttons without periods?
- [ ] Are numbers formatted consistently?
- [ ] Are times and dates in Russian standard format?
